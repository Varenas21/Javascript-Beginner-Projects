Welcome to Basic Tools!

Basic Tools is a beginner-level Javascript website that combines three interactive mini-projects into a single platform. Each project is designed to demonstrate specific Javascript concepts such as working with libraries, manipulating objects, and using operators. This project also highlights my ability to structure and style responsive web pages using HTML and CSS, integrating them with dynamic Javascript functionality. Together, these tools reflect my growing understanding of front-end development and my ability to build functional, user-friendly interfaces. Below is a breakdown of the key concepts each tool focuses on:

Functional Calculator

This calculator project teaches me how to use Javascript operators and handle user input through buttons. It supports basic arithmetic, including modulus, and reinforces logic building and real-time calculations.

To-Do List

In this project, I'm focusing on Javascript objects and DOM manipulation. Users can add, delete and organize tasks, which helps me handling dynamic data and creating interactive user interfaces.

Timer & StopWatch

This project helps me learn how to use Javascript libraries and manage time-based functions. It includes a customizable timer and a stopwatch with start, stop and reset functionality, along with alerts to notify when time is up.
